Exploring different type of fonts

Serif have small lines at the ends of some characters to be stylish.

  • Times New Roman
  • Georgia

Sans-Serif fonts do not. "Sans" means without.

  • Arial
  • Verdana

Monospace fonts ensure that all characters have the same space (width).

  • Courier
  • Lucida Console

How do we read this?

font-family: "Arial", Arial, sans-serif;

If you don't have this specific font, then it will default to the other fonts you set after it. Basically a backup.

Suppose we want MORE CHOICES

Google has us covered as all we need to do is to link a font tag with the font-family name after the question mark.

Ex:
link https://fonts.googleapis.com/css?family=Sofia' rel=stylesheet
in angle brackets in the head section, this would give you the font-family of sofia.
